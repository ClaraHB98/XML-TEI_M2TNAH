# Utiliser CollateX

- Création d'un environnement virtuel
``` bash
python3 -m venv venv
```
- Activatuon de l'environnement
``` bash
source venv/bin/activate
```
- Installation des dépendances
``` bash
python3 -m pip install -r requirements.txt
```
- Lancement du script
``` bash
python3 collation.py
```
